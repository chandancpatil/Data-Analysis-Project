# E-Commerce Store Analysis (Power BI Dashboard)

## 📊 Project Overview
This project presents an interactive Power BI dashboard built to analyze the performance of an E-commerce store. The dashboard helps explore sales trends, customer behavior, and product performance using visual analytics to support data-driven business decisions.

## 🚀 Key Features
- Interactive dashboard to track and analyze **online sales data**
- Drill-down analysis using **filters, slicers, and parameters**
- Visual representation of **sales trends, categories, and customer insights**
- Multiple customized visualizations for better data understanding

## 🛠 Tools & Technologies
- Microsoft Power BI
- DAX
- Excel (Dataset)

## 📂 Dataset
The dataset contains information related to:
- Customer details  
- Product categories  
- Sales transactions  
- Order and revenue data  

## 📈 Project Learnings
- Built an interactive **Power BI dashboard** for sales analysis
- Created **relationships and joins between tables**
- Performed **data transformation and calculations using DAX**
- Used multiple visualizations such as **bar charts, pie charts, line charts, maps, and slicers**

## 📊 Insights
The dashboard provides insights such as:
- Sales performance across different product categories
- Customer purchase trends
- Revenue distribution across regions
- Identification of high-performing products

## 🔮 Future Improvements
- Add real-time sales data integration
- Include advanced KPIs and forecasting
- Enhance dashboard interactivity for deeper analysis
